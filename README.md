## Installatie
1. **Clone de repository:**  
   ```bash
   git clone https://github.com/RickDortland/Narco-Ninja-s.git
   cd narco-ninja-s
   ```
2. **Installeer de dependencies:**  
   ```bash
   npm install
   ```
3. **Start de ontwikkelserver:**  
   ```bash
   npm run dev
   ```
4. De applicatie draait nu op `http://localhost:3000`.
